[UTKAST - BEHÖVER GRANSKAS]

Det finns flera förklaringar till varför kortison inte hjälpte denna gång.

### Möjliga orsaker

| Orsak | Förklaring |
|-------|------------|
| Fel injektionsställe | Möjligt utan ultraljud |
| Annan smärtorsak | Inte inflammatorisk |
| Sjukdomsprogression | Artrosen har förvärrats |
| Effekten tar tid | Kan ta 1-2 veckor |

### Ultraljudsvägledning

| Faktor | Med ultraljud | Utan ultraljud |
|--------|---------------|----------------|
| Precision | Hög | Lägre |
| Terapeutisk effekt | Ofta bättre | Varierar |

### Vad kan du göra?

| Åtgärd | Kommentar |
|--------|-----------|
| Vänta 1-2 veckor | Ibland fördröjd effekt |
| Återbesök | Om ingen förbättring |
| Begär ultraljudsväglett | Vid nästa injektion |
| Utredning | Om upprepad svikt |

### Rekommendation

**Om tidigare injektioner hjälpt och denna inte gör det, är det rimligt att:**
1. Vänta ytterligare några dagar
2. Återkomma om ingen effekt
3. Begära ultraljudsvägledning nästa gång

**Sammanfattning:** Din misstanke kan stämma – ultraljudsvägledning ökar precisionen.
