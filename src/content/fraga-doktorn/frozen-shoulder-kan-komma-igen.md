[UTKAST - BEHÖVER GRANSKAS]

Frozen shoulder efter operation kan teoretiskt återkomma, men det är **ovanligt**.

### Recidivrisk

| Situation | Risk |
|-----------|------|
| Samma axel efter operation | <5% |
| Andra axeln | 5–15% (du har redan haft) |
| Tredje gången | Mycket ovanligt |

### Riskfaktorer för ny frozen shoulder

| Faktor |
|--------|
| Diabetes |
| Sköldkörtelsjukdom |
| Hjärt-kärlsjukdom |
| Tidigare trauma/operation |

### Hur känner man igen det?

| Symtom |
|--------|
| Tilltagande stelhet |
| Smärta, särskilt nattetid |
| Svårt att röra armen åt sidan |

### Vad kan du göra?

| Åtgärd | Syfte |
|--------|-------|
| Fortsätt röra axlarna | Förebyggande |
| Tidig behandling | Om symtom börjar |
| Kontroll av grundsjukdom | Diabetes etc. |

**Sammanfattning:** Risken att få frozen shoulder igen i samma axel efter operation är låg (<5%). Fortsätt använda axlarna normalt.
