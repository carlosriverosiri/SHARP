question: |
  Jag ramlade i skidbacken så att mitt vänstra knä böjdes kraftigt och senare fick jag lätt ont på insidan av knät, den smärtan försvann helt efter 2 dagar dock. Ett problem som dök upp senare var att ibland låser sig knät, för övrigt har jag inga problem att böja, sträcka eller hoppa. Vad kan det vara för fel på mitt knä?

[UTKAST - BEHÖVER GRANSKAS]

Låsningar efter trauma tyder ofta på meniskskada.

### Typiska orsaker till knälåsning

| Orsak | Sannolikhet |
|-------|-------------|
| Meniskskada | Hög |
| Lös kropp | Möjlig |
| Broskskada | Möjlig |
| Främre korsbandsskada | Möjlig |

### Din situation

| Faktor | Betydelse |
|--------|-----------|
| Trauma (skidfall) | Typisk skademekanism |
| Initial smärta försvann | Vanligt vid meniskskada |
| Låsningar kvarstår | Tyder på mekaniskt hinder |
| Funktion i övrigt ok | Positivt |

### Utredning

| Undersökning | Syfte |
|--------------|-------|
| Klinisk undersökning | Menisktest, stabilitet |
| MR | Bekräfta diagnos |

### Behandling av menisklåsning

| Alternativ | Indikation |
|------------|------------|
| Sjukgymnastik | Lindriga besvär |
| Artroskopi | Vid upprepade låsningar |

### Rekommendation

**Upprepade låsningar bör utredas.** Om knät låser sig så att du inte kan sträcka eller böja det fullt ut, talar detta för meniskskada som kan behöva opereras.

**Sammanfattning:** Sök bedömning – låsningar efter trauma bör undersökas med MR.
