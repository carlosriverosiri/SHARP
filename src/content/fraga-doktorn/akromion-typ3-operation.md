  Jag har haft problem med min högra axel i ett år. Jag har provat sjukgymnast, kortisonsprutor, akupunktur, även med el – ingenting har fungerat. [UTKAST - BEHÖVER GRANSKAS]

Akromion typ 3 (kroknäbb) kan bidra till impingement.

### Akromiontyper

| Typ | Form | Impingement-risk |
|-----|------|------------------|
| Typ 1 | Platt | Låg |
| Typ 2 | Böjd | Måttlig |
| Typ 3 | Krokig | Hög |

### Din situation

| Faktor | Status |
|--------|--------|
| Duration | 1 år ✓ |
| Konservativ behandling | Prövad ✓ |
| Kortisonsprutor | Hjälpte inte ✓ |
| Funktionspåverkan | Ja ✓ |

### Operation – akromioplastik

| Aspekt | Beskrivning |
|--------|-------------|
| Ingrepp | Titthålskirurgi, hyvla akromion |
| Sjukhusvistelse | Dagkirurgi |
| Rehabilitering | 3-6 månader |
| Framgångsfrekvens | Cirka 80% |

### Rekommendation

**Med 1 års besvär, misslyckad konservativ behandling och typ 3-akromion är operation rimligt att diskutera.** 

Men: Först bör man utesluta rotatorkuffskada med MR eller ultraljud.

**Sammanfattning:** Operation kan hjälpa – men gör MR först för att se hela bilden.
