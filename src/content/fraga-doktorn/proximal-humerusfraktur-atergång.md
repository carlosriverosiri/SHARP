question: |
  I början på april ramlade jag ner från en stege och bröt min axel (proximal humerusfraktur). Jobbar med yngre barn på förskola. Min fråga är hur lång tid tar det innan man kan återgå till arbete? I detta fall tungt arbete med många lyft.
---

[UTKAST - BEHÖVER GRANSKAS]

Proximal humerusfraktur kräver tid för läkning, särskilt vid tungt arbete.

### Tidsram för läkning

| Fas | Tid |
|-----|-----|
| Benläkning | 6–8 veckor |
| Rörlighetsträning | 8–12 veckor |
| Styrketräning | 12–16 veckor |
| Full styrka | 4–6 månader |

### Återgång till förskolearbete

| Arbetsmoment | Tid |
|--------------|-----|
| Administrativa uppgifter | 6–8 veckor |
| Lätt arbete (ej lyft) | 8–10 veckor |
| Gradvis ökande lyft | 12–16 veckor |
| Fullt arbete | 4–5 månader |

### Faktorer som påverkar

| Faktor | Inverkan |
|--------|----------|
| Frakturens typ | Komminuta tar längre tid |
| Din ålder (38) | God läkningsförmåga |
| Rehabilitering | Avgörande |
| Opererad eller ej | Kan påverka tidplanen |

### Rekommendation

| Steg | Åtgärd |
|------|--------|
| 1 | Följ sjukgymnastens program |
| 2 | Gradvis återgång med arbetsanpassning |
| 3 | Undvik tunga lyft första 3 månaderna |

**Sammanfattning:** Räkna med 4–5 månader innan du kan lyfta barn som tidigare. Diskutera arbetsanpassning med din arbetsgivare.
