  Opererades för tre veckor sedan, ruptur av degenerativ supraspinatus. Sedan dess ständig smärta. Är det normalt och hur länge kommer jag att ha smärta?

[UTKAST - BEHÖVER GRANSKAS]

**Ja, det är normalt med smärta 3 veckor efter cuffoperation.**

### Typiskt smärtförlopp

| Tid | Smärtnivå |
|-----|-----------|
| 0-2 veckor | Kraftig |
| 2-6 veckor | Måttlig, avtagande |
| 6-12 veckor | Mild |
| 3-6 månader | Minimal |

### Varför gör det ont?

| Orsak | Förklaring |
|-------|------------|
| Operationstrauma | Normal läkningsprocess |
| Immobilisering | Stelhet ger smärta |
| Inflammation | Läkningssvar |

### Smärtlindring

| Åtgärd | Kommentar |
|--------|-----------|
| Ordinerad smärtstillning | Följ instruktioner |
| Is | 15-20 min x flera gånger |
| Sova halvliggande | Minskar nattsmärta |
| Mitella | Avlastar |

### När ska du oroa dig?

| Varningstecken | Åtgärd |
|----------------|--------|
| Feber | Kontakta kliniken |
| Rodnad och svullnad vid såret | Kontakta kliniken |
| Domningar/stickningar | Kontakta kliniken |

**Sammanfattning:** Smärta 3 veckor efter cuff-op är helt normalt. Räkna med gradvis förbättring över 2-3 månader.
