question: |
  Har artros i axelled. Väldiga problem vid träning och på jobbet. Elisabethsjukhuset sa att protes är enda räddningen men de avrådde pga min relativt unga ålder. Om jag gör operationen, hur blir mitt liv då? Tänker på träning samt jobb som elektriker och MTB-cykling.
---

Förståelig oro. Vid 46 år med fysiskt arbete är beslutet om axelprotes en avvägning.

### Liv efter axelprotes

| Aktivitet | Förväntat resultat |
|-----------|-------------------|
| Smärta | Kraftigt minskad |
| Rörlighet | Förbättrad, men inte alltid full |
| Styrka | God för de flesta aktiviteter |
| Sömn | Betydligt bättre |

### Arbete som elektriker

| Aspekt | Bedömning |
|--------|-----------|
| Lätt arbete | Inga problem |
| Arbete i axelhöjd | Ofta möjligt |
| Tunga lyft över huvudet | Begränsas |
| Vibrationsverktyg | Viss försiktighet |

Många med fysiska yrken återgår till arbete, men anpassningar kan behövas.

### Träning och MTB

| Aktivitet | Efter protes |
|-----------|--------------|
| MTB-cykling | Ja, de flesta |
| Gym/styrketräning | Ja, anpassat |
| Simning | Utmärkt |
| Kontaktsport | Nej |

### Problemet med ung ålder

| Utmaning | Konsekvens |
|----------|------------|
| Lång livstid | Protesen kan behöva bytas |
| Hög aktivitet | Snabbare slitage |
| Revision | Svårare operation |

### Alternativ att överväga

| Behandling | Kan ge tid |
|------------|------------|
| Kortison | Månader |
| Hyaluronsyra | Oklart |
| Anpassad träning | Varierande |
| Artroskopi | Begränsad nytta vid artros |

### Min rekommendation

1. **Optimera konservativ behandling** – Sjukgymnastik, kortison
2. **Acceptera eventuell operation** – När smärtan överstiger riskerna
3. **Välj erfaren kirurg** – Resultaten varierar
4. **Var realistisk** – Du kommer kunna cykla och jobba, men med viss anpassning

**Sammanfattning:** Axelprotes vid 46 är inte optimalt men kan vara rätt beslut om smärtan är för stor. De flesta återfår god funktion och kan fortsätta arbeta och träna.
