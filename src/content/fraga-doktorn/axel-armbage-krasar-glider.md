question: |
  Min höger axel och armbåge började ganska plötsligt krasa ganska ordentligt. Dessutom känns det efter det som om armbågen ibland glider lite "ur led" och hoppar tillbaka i läge igen lika snabbt. Jag har ingen smärta i varken axel eller armbåge. Kan belasta armbågen, tex lyfta tunga kassar, men inte göra rörelser med tyngden uppifrån, tex armhävningar utan att det krasar som bara den. Vad kan detta vara?

[UTKAST - BEHÖVER GRANSKAS]

Krasande leder utan smärta är vanligare än man tror.

### Möjliga orsaker till krasandet

| Orsak | Förklaring |
|-------|------------|
| Senor glider | Vanligaste orsaken |
| Broskytor | Kan ge ljud |
| Luft i leden | Harmlöst |
| Överrörlighet | Kan ge "glid"-känsla |

### Armbågens "glidkänsla"

| Orsak | Sannolikhet |
|-------|-------------|
| Subluxation | Möjligt |
| Sensnapping | Troligt |
| Nervpåverkan | Mindre troligt |

### När bör du söka?

| Situation | Åtgärd |
|-----------|--------|
| Ingen smärta, ingen svaghet | Avvakta |
| Smärta tillkommer | Sök bedömning |
| Svaghet utvecklas | Sök bedömning |
| Låsningar | Sök bedömning |

### Rekommendation

**Krasande utan smärta eller funktionsnedsättning kräver inte akut åtgärd.** Men:
- Om "glid"-känslan blir mer uttalad
- Om smärta tillkommer
- Då bör du söka ortoped

**Sammanfattning:** Troligen ofarligt, men följ utvecklingen. Sök om symtomen förvärras.
