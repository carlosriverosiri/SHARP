question: |
  Har smärta strax medialt om tuberositas tibia, ca 0,5 kvcm yta. Finns hela tiden, påverkar inte löpningen nämnvärt. Gjort uppehåll i löpningen och alternativtränar nu. Känner inget vid gång men fortfarande palpationsömhet. Ingen rodnad eller svullnad. Bör man kolla om det kan röra sig om stressfraktur? Patellarseneskada? Annat?

Som kollega kan jag resonera lite mer ingående kring differentialdiagnostiken.

### Differentialdiagnoser

| Diagnos | Talar för | Talar emot |
|---------|-----------|------------|
| Stressfraktur tibia | Palpationsömhet, löpning | Lokalisationen, ingen svullnad |
| Pes anserinustendinit | Medialt, löpare | Ofta mer diffust |
| Patellarsenetendinit | Nära tuberositas | Oftast över senan |
| Benhinneinflammation | Löpning | Vanligen längs tibiakanten |

### Stressfraktur – sannolikhet?

| Faktor | Din situation |
|--------|---------------|
| Lokalisation | Atypisk för tibiastress |
| Smärtprofil | Inte klassisk (nattsmärta saknas?) |
| Fysundersökning | Begränsad ömhet |
| Riskfaktorer | Ålder, löpning |

### Behöver du utreda?

| Undersökning | Indikation |
|--------------|------------|
| Röntgen | Låg sensitivitet tidigt |
| MR | Guldstandard för stressfraktur |
| Ultraljud | Bra för mjukdelar, sämre för ben |

### Min bedömning

Med lokaliserad ömhet, avsaknad av svullnad och smärta som inte påverkar löpningen nämnvärt bedömer jag stressfraktur som **mindre sannolikt**. Mer troligt är en lokal tendinopati eller insertit.

### Förslag

| Åtgärd | Motivering |
|--------|------------|
| Fortsätt alternativträning | 2–4 veckor |
| Excentrisk träning | Om misstänkt tendinopati |
| MR | Om kvarstående symtom >6 veckor |
| Klinisk uppföljning | Vid försämring |

**Sammanfattning:** Med din symtombild är stressfraktur mindre sannolikt. Avvakta med bilddiagnostik om du förbättras, men MR är motiverat vid kvarstående besvär.
