question: |
  Jag undrar om ett fall på axel kan orsaka impingement. Eller är det enbart en förslitnings-skada eller skulderledssjukdom? Jag har fått berättat för mig att det inte kan bero på fall.
---

[UTKAST - BEHÖVER GRANSKAS]

**Ja, ett fall kan absolut bidra till impingement.**

### Orsaker till impingement

| Typ | Beskrivning |
|-----|-------------|
| Strukturellt | Kroknäbb (akromion typ 3) |
| Förslitning | Gradvis senskada |
| Traumatiskt | Efter fall eller slag |
| Kombinerat | Vanligast |

### Hur trauma kan orsaka impingement

| Mekanism | Förklaring |
|----------|------------|
| Svullnad | Akut inflammation minskar utrymmet |
| Senblödning | Gör senan tjockare |
| Partiell ruptur | Nedsatt funktion |
| Ändrad biomekanik | Skyddsreaktion |

### Praktisk betydelse

| Fråga | Svar |
|-------|------|
| Arbetsskada? | Trauma kan vara utlösande |
| Försäkring? | Dokumentera tidssambandet |
| Behandling? | Samma oavsett orsak |

### Rekommendation

Det går absolut att få impingement efter ett trauma. **Dokumentera händelsen** om det är relevant för försäkring eller arbetsskada.

**Sammanfattning:** Impingement kan vara både degenerativt och traumatiskt – ofta en kombination.
