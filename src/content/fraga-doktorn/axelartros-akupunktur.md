[UTKAST - BEHÖVER GRANSKAS]

Akupunktur är en vanlig fråga vid kronisk smärta.

### Vad säger evidensen?

| Faktor | Bedömning |
|--------|-----------|
| Vetenskapligt stöd | Begränsat för axelartros |
| Smärtlindring | Viss effekt hos några |
| Biverkningar | Minimala |
| Kostnad | Ej subventionerat |

### Behandlingsalternativ vid axelartros

| Behandling | Effekt | Kommentar |
|------------|--------|-----------|
| Sjukgymnastik | Bra | Förstahandsval |
| NSAID | Måttlig | Tillfällig lindring |
| Kortison | Tillfällig | 4-12 veckors effekt |
| Protes | Bra | Vid svår artros |
| Akupunktur | Okänd | Kan provas |

### Praktiskt råd

Det är inte fel att prova akupunktur, men:
- Förvänta dig inte mirakel
- Kombinera med sjukgymnastik
- Överväg kortisoninjektion för snabbare lindring

### Rekommendation

**Vid "fruktansvärd värk" av axelartros bör du träffa ortoped.** Akupunktur kan ge viss lindring men löser inte grundproblemet.

**Sammanfattning:** Akupunktur kan provas men är ingen huvudbehandling vid svår axelartros.
