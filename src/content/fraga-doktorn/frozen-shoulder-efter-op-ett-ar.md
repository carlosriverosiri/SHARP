[UTKAST - BEHÖVER GRANSKAS]

Frozen shoulder efter operation kan ta längre tid att läka.

### Typiskt förlopp

| Fas | Tid |
|-----|-----|
| Frysning | 3–9 månader |
| Frusen | 4–12 månader |
| Upptining | 6–24 månader |

### Efter 1 år

| Prognos |
|---------|
| De flesta förbättras gradvis |
| Kan ta upp till 2–3 år |
| Viss kvarstående stelhet möjlig |

### Rekommendation

| Åtgärd |
|--------|
| Fortsätt sjukgymnastik |
| Överväg ny bedömning |
| Tålamod – det blir bättre |

**Sammanfattning:** Det kan ta upp till 2 år. Fortsätt rehabilitera.
