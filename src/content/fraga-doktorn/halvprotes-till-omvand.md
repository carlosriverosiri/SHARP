[UTKAST - BEHÖVER GRANSKAS]

Revision till omvänd protes kan ge bra resultat.

### Vanliga orsaker till revision

| Orsak | Förekomst |
|-------|-----------|
| Cuffskada efter op | Vanlig |
| Instabilitet | Möjlig |
| Fortsatt artros | Möjlig |

### Prognos för omvänd protes efter halvprotes

| Faktor | Förväntning |
|--------|-------------|
| Smärtlindring | God |
| Rörlighet | Ofta förbättrad |
| Kraft | Kan förbättras |
| Komplikationsrisk | Något högre än primär |

**Sammanfattning:** Omvänd protes efter misslyckad halvprotes kan ge gott resultat, om än med lite högre risk.
