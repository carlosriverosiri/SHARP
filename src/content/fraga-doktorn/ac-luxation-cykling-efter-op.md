[UTKAST - BEHÖVER GRANSKAS]

Cykling på trainer är en bra fråga efter AC-operation.

### Typisk tidsplan

| Vecka | Aktivitet |
|-------|-----------|
| 0-2 | Endast armpendling |
| 2-4 | Lätt promenad |
| **4-6** | **Försiktig indoorcykling** |
| 6-12 | Gradvis ökad belastning |

### Förutsättningar för cykling

| Villkor | Kommentar |
|---------|-----------|
| Ingen smärta i vila | Grundkrav |
| OK från operatör | Fråga din läkare |
| Upprätt position | Ingen aerobar |
| Ingen belastning på armen | Händerna vilar lätt |

### Praktiska tips

| Tips | Varför |
|------|--------|
| Mjuk sadel | Undvik vibrationer |
| Upprätt cykel | Mindre tryck på axeln |
| Korta pass först | 15-20 min |
| Avbryt vid smärta | Lyssna på kroppen |

### Rekommendation

**Omkring 4-6 veckor efter operation kan du vanligen börja cykla försiktigt inomhus** – förutsatt att:
- Du inte belastar armen
- Du sitter upprätt
- Operatören godkänt det

**Sammanfattning:** Fråga din opererande läkare, men 4-6 veckor är ofta rimligt för lugn indoorcykling.
