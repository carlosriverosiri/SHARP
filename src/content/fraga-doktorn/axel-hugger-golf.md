question: |
  Hugger till i axeln vid viss rörelse uppåt. Känns som axeln går ur led. Varit hos sjukgymnast som trodde att det var vid muskelfästet. Skulle smörja med Voltaren och gummibandsövningar. Har inte blivit bättre. Vad kan det vara?
---

En huggande smärta vid upplyft arm som **inte blivit bättre** av Voltaren och träning kan ha flera orsaker.

### Möjliga diagnoser

| Diagnos | Typiska tecken |
|---------|----------------|
| Impingement | Smärta 60–120° upplyft |
| Rotatorkuffskada | Svaghet, nattsmärta |
| Kalkaxel | Plötslig intensiv smärta |
| AC-ledsartros | Smärta vid arm tvärs kroppen |
| Bicepstendinit | Smärta fram i axeln |

### Varför hjälpte inte behandlingen?

| Möjlig förklaring |
|-------------------|
| Fel diagnos |
| Strukturell skada som inte läker |
| För kort behandlingstid |
| Behöver annan behandling |

### Vad bör du göra?

| Steg | Åtgärd |
|------|--------|
| 1 | Ny läkarbedömning |
| 2 | Ultraljud eller MR |
| 3 | Anpassa behandlingen |

### Behandlingsalternativ beroende på diagnos

| Diagnos | Behandling |
|---------|------------|
| Impingement | Kortisoninjektion, operation |
| Cuffskada | Kirurgi vid stor ruptur |
| Kalkaxel | Nålning, stötvåg, operation |
| AC-artros | Injektion, ev. operation |

### Min rekommendation

1. **Boka läkartid** – Sjukgymnastdiagnos kan behöva bekräftas
2. **Bilddiagnostik** – Ultraljud är snabbt och ger svar
3. **Överväg injektion** – Om inflammation misstänks
4. **Golf** – Anpassa tills diagnos ställts

**Sammanfattning:** Tre månader utan förbättring motiverar ny bedömning. En strukturell skada kan kräva annan behandling än enbart träning.
