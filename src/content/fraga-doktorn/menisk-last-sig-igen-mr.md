  Jag har problem med menisken. Har en gammal skidskada på vänster knä som jag drog upp i januari när jag fick ett kraftigt felskär på långfärdsskridskor. Har kört rehab med bra resultat och kände mig nästan helt återställd. I lördags när jag trampade vatten så vred jag på något sätt och mitt knä låste sig. Finns det möjlighet att få magnetröntgen hos er och eventuell operation? Vad skulle det kosta? Har ingen privat sjukförsäkring.

Med en knälåsning efter tidigare meniskskada bör du definitivt bli undersökt.

### Ja, vi kan hjälpa dig

| Tjänst | Finns hos oss |
|--------|---------------|
| Bedömning | Ja |
| Ultraljud | Ja (begränsat för menisk) |
| MR | Remiss till extern klinik |
| Artroskopi | Ja |

### Prisöversikt (utan försäkring)

| Tjänst | Ungefärligt pris |
|--------|------------------|
| Första besök | ca 2 500 kr |
| MR (extern) | ca 3 000–4 000 kr |
| Artroskopi menisk | ca 25 000–35 000 kr |
| Efterkontroll | Ingår |

### Knälåsning – vad betyder det?

| Tecken | Betydelse |
|--------|-----------|
| Mekanisk låsning | Något sitter i kläm |
| Kan inte sträcka helt | Vanligt vid meniskflip |
| Plötslig debut | Typiskt för menisk |

### Behöver du operation?

| Situation | Rekommendation |
|-----------|----------------|
| Kvarstående låsning | Operation motiverad |
| Återkommande låsningar | Operation motiverad |
| Engångshändelse, nu bra | Avvakta, rehabilitera |

### Alternativ väg

| Alternativ | Fördelar | Nackdelar |
|------------|----------|-----------|
| Via husläkare | Subventionerat | Längre väntetid |
| Egenremiss till oss | Snabbt | Egen kostnad |
| Akutmottagning | Om akut låst | Endast akut |

### Min rekommendation

1. **Boka tid hos oss** – För bedömning
2. **Ta med** – Eventuella tidigare röntgenbilder/journaler
3. **Vi planerar** – MR vid behov, sedan behandling

**Sammanfattning:** Vi kan hjälpa dig med hela kedjan från bedömning till eventuell operation. Boka tid så undersöker vi knät.
