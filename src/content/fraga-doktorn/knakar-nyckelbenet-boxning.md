Knakande ljud från sternoclavicularleden (SC-leden, där nyckelbenet möter bröstbenet) är **vanligt och oftast ofarligt**.

### Vad orsakar ljudet?

| Orsak | Beskrivning |
|-------|-------------|
| Gasbubblor | Som vid knäppande fingrar |
| Sena glider | Normalt fenomen |
| Mindre ojämnheter | I ledytan |
| Hypermobilitet | Vanligt hos unga |

### När är det ofarligt?

| Tecken på normalitet |
|---------------------|
| Ingen smärta |
| Ingen svullnad |
| Full rörlighet |
| Stabil led |

Du uppfyller alla dessa kriterier.

### När bör man söka vård?

| Varningssignal |
|----------------|
| Smärta vid knakandet |
| Svullnad över leden |
| Synlig förändring/knöl |
| Instabilitet |
| Knakandet efter trauma |

### Boxning och SC-leden

SC-leden utsätts för krafter vid slag och blockeringar. Så länge du inte har smärta finns ingen anledning att sluta träna.

### Min rekommendation

| Åtgärd | Motivering |
|--------|------------|
| Fortsätt träna | Inga tecken på problem |
| Observera | Om smärta tillkommer – sök vård |
| Styrketräning | Stärker omgivande muskulatur |

**Sammanfattning:** Knakande utan smärta är normalt och ofarligt. Fortsätt boxas!
